The Pipeline

Six steps, listing to inbox

Every week the system pulls fresh listings, filters them against your brief, scores the survivors, and delivers a ranked shortlist straight to your email. Here's how each stage works.

1
Sources
Pull listings from multiple property sites across 26 postcodes in the Central and Southern Tablelands. Cross-referencing sources catches properties that only appear on one platform.
• Domain.com.au (web scraper)
• REA / realestate.com.au (via Apify)
• Elders Real Estate (web scraper)
• Southern Tablelands Realty (web scraper)
• Farmbuy.com (web scraper)
• CRE.com.au email alerts
• Nutrien Harcourts alerts
• AuctionsPlus (registered)
• Listing Loop (off-market)
• Property Whispers (off-market)
5 active scrapers 26 postcodes 10 channels
2
Normalize
Every source uses different formats. This step converts them all into a common structure — address, price, acreage, bedrooms, coordinates, photos, URL — and deduplicates across sources so the same property isn't counted twice.
Common format Deduplicated
3
Gate Filter
Hard pass/fail criteria. A listing must clear every gate or it's silently discarded. No partial credit — these are the non-negotiables from your brief.
• Geography (13 target LGAs)
• Budget ($100K–$2M)
• Land size (30–200 acres)
• Existing house
• Rural zoning
• Drive time (≤4 hours)
163 pass 49% pass rate
4
Enrich
Search pages only show basic data. For every property that passes the gates, the system fetches the full listing page to pull the complete description, headline, and agent details. This is what makes accurate scoring possible.

Feedback Loop

Results sharpen over time

Every reaction you tap — Love it, Interesting, Not for me — gets logged. Over time, the system learns what you actually care about beyond what the brief says on paper. If you keep loving properties with creek frontage and ignoring ones near towns, the weights shift to match.

Feedback is captured via one-click links in the email, logged to a private Google Sheet. No apps, no logins, no friction.
